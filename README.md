# Patrones de Diseño en Java

Este repositorio contiene ejemplos prácticos y ejercicios sobre **Patrones de Diseño** implementados en Java. El objetivo es comprender y aplicar los patrones de diseño más comunes en el desarrollo de software, siguiendo las mejores prácticas de arquitectura de software.

##  ¿Qué son los Patrones de Diseño?

Los patrones de diseño son soluciones reutilizables a problemas comunes en el diseño de software. Proporcionan plantillas probadas que ayudan a crear código más flexible, mantenible y escalable.

##  Contenido del Repositorio

Este repositorio está organizado por carpetas, cada una enfocada en un patrón específico:

### 1. Singleton Pattern
- **[1.SingletonExample](./1.SingletonExample/)** - Implementación básica del patrón Singleton
- **[2.SingletonThreadSafe](./2.SingletonThreadSafe/)** - Implementación thread-safe del patrón Singleton
- **[3.singleton-Ejercicio](./3.singleton-Ejercicio/)** - Ejercicio práctico: Calculadora Singleton
- **[4.Singleton-Ejercicio2](./4.Singleton-Ejercicio2/)** - Ejercicio práctico: Sistema de Sesión

### 2. Factory Pattern
- **[5.factory](./5.factory/)** - Implementación del patrón Factory Method

## Objetivo de Aprendizaje

A través de estos ejemplos, se busca:
- Entender cuándo y cómo aplicar cada patrón de diseño
- Implementar soluciones elegantes a problemas recurrentes
- Mejorar la calidad y mantenibilidad del código
- Desarrollar habilidades de diseño orientado a objetos

## Tecnologías

- **Lenguaje**: Java
- **IDE**: Visual Studio Code
- **Paradigma**: Programación Orientada a Objetos
